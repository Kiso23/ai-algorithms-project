## 📖 ALGORITHM DESCRIPTIONS

### 1. Alpha-Beta Pruning
**Purpose**: Optimize minimax algorithm for game tree search

**Key Features**:
- Eliminates branches that don't affect final decision
- Complexity: O(b^d) → O(b^(d/2)) with pruning
- Tracks nodes explored and pruned
- Shows efficiency metrics

**Demo**: Shows game tree traversal with pruning in action

### 2. Hidden Markov Model (Viterbi Algorithm)
**Purpose**: Find most likely hidden state sequence given observations

**Key Features**:
- Viterbi algorithm for optimal state sequence
- Forward algorithm for probability calculation
- Backward algorithm for state estimation
- Weather prediction example

**Demo**: Predicts weather patterns from observations

### 3. Tic-Tac-Toe with AI
**Purpose**: Unbeatable game opponent using minimax + alpha-beta pruning

**Key Features**:
- Complete game logic with win detection
- Minimax search with alpha-beta pruning
- Real-time search statistics
- Interactive command-line gameplay

**Demo**: Play against intelligent AI opponent

## 📈 ALGORITHM COMPLEXITY

| Algorithm | Time Complexity | Space Complexity |
|-----------|-----------------|------------------|
| Alpha-Beta Pruning | O(b^(d/2)) best | O(d) |
| Viterbi Algorithm | O(T × N²) | O(T × N) |
| Tic-Tac-Toe AI | O(n) where n~100K | O(d) |

Where:
- b = branching factor
- d = tree depth
- T = time steps
- N = number of states
- n = nodes explored
